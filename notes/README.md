## Favicons
The favicons are generated with [this](https://realfavicongenerator.net/) page. They are stored at `/media/icons/favicon/`.

[Purpose](https://webflow.com/blog/favicon-guide) of these generated icons should be to increase accessability by providing multiple sizes of web favicon.

The `.ico` files can be checked for sizes of icons that are contained in that file with [this online tool](https://redketchup.io/).


## Schemas (For web pages)

Schemas are structured mark-up data on websites, used (primarily) by search engines to create [rich results](https://www.schemaapp.com/schema-markup/what-is-a-rich-result/). This allows for better presence when the page is searched, can improve rankings, and helps web crawlers get useful information about the page.

Schemas in portfolio portion:
 - [Person schema](https://schema.org/Person)
 - [WebSite schema](https://schema.org/WebSite)
